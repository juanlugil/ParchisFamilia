# Parchís Familiar

Juego de parchís personalizable para la familia usando HTML/CSS/JS.

## Cómo usar

1. Abre `index.html` en un navegador moderno.
2. Carga las imágenes de los 4 jugadores.
3. Haz clic en `Iniciar partida`.
4. Presiona `Lanzar dado` para ver el resultado de cada turno.

## Características actuales

- Tablero para 4 jugadores con colores tradicionales.
- Cada jugador tiene 4 fichas.
- Las fichas muestran una imagen personalizada en el centro.
- El primer jugador se elige aleatoriamente.
- Turno siguiente en sentido horario.
- Para sacar ficha de casa se requiere un 5.
- Un 6 permite mover y tirar de nuevo; el tercer 6 obliga a devolver una ficha a casa.
- Al entrar en la meta se concede un avance adicional de 10 casillas con otra ficha.
- Al capturar una ficha rival, la ficha capturada vuelve a casa y se puede avanzar otra ficha 20 casillas.
- Las fichas pueden completar el recorrido y llegar a la meta final.
- Detección de ganador y fin de partida cuando un jugador termina sus 4 fichas.
- Dado virtual animado durante 5 segundos.
